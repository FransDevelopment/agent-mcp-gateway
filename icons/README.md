# Extension Icons

Chrome Web Store requires icons at these sizes:
- `icon-16.png` — toolbar (16x16)
- `icon-48.png` — extensions page (48x48)
- `icon-128.png` — Chrome Web Store listing (128x128)

Additionally, the store requires:
- A promotional tile (440x280 or 920x680)
- At least 1 screenshot (1280x800 or 640x400)

Generate these from the Arcede brand assets before publishing.
